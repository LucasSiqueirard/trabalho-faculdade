Ferramentas e Tecnologias Utilizadas

A seguir estão os recursos utilizados no desenvolvimento do site, juntamente com uma breve explicação sobre o que cada um faz.

ChatGPT

  • O que é: Uma inteligência artificial avançada criada pela OpenAI para gerar texto, responder perguntas e auxiliar em tarefas.
  • Para que serve: Foi usado para criar textos, explicações, estruturação de conteúdo e apoio na construção do código HTML e CSS.

Gemini

  • O que é: Uma inteligência artificial avançada criada pela Google para gerar texto, responder perguntas e auxiliar em tarefas.
  • Para que serve: Foi usado para criar textos, explicações, estruturação de conteúdo e apoio na construção do código HTML, CSS e JavaScript

Visual Studio Code (VS Code)

  • O que é: Um editor de código leve e poderoso, amplamente usado por desenvolvedores.
  • Para que serve: Utilizado para escrever, organizar e editar os arquivos do site (HTML, CSS e JavaScript).

GitHub

  • O que é: Uma plataforma de hospedagem de repositórios que utiliza Git para versionamento de código.
  • Para que serve: Serve como armazenamento online do projeto, permitindo salvar, organizar e compartilhar o código.

Git

  • O que é: Um sistema de controle de versão que registra alterações nos arquivos ao longo do tempo.
  • Para que serve: Permite controlar as versões do site, acompanhar modificações, recuperar versões antigas e trabalhar em equipe com segurança.

HTML (HyperText Markup Language)

  • O que é: A linguagem de marcação usada para estruturar páginas da web.
  • Para que serve: Criou toda a base estrutural do site, como títulos, textos, seções e organização do conteúdo.

CSS (Cascading Style Sheets)

  • O que é: Linguagem responsável pela estilização e aparência visual das páginas.
  • Para que serve: Usado para aplicar cores, fontes, tamanhos, alinhamentos e deixar o site visualmente agradável.

JavaScript

  • O que é: Uma linguagem de programação usada para adicionar interatividade às páginas web.
  • Para que serve: Pode ser utilizado para criar animações, validações, menus dinâmicos e funcionalidades interativas no site.
